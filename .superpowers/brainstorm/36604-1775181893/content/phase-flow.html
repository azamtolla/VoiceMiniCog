Key Design Rules

  • Avatar panel: Always 60% width, right side, black background, rounded corners
  • Left panel: 40% width, adapts per phase. Minimal = phase name + progress + content. Expanded = interactive element fills the space.
  • Expanded phases: Only Clock Drawing and Verbal Fluency. Border color changes to phase accent color.
  • Transitions: Left panel content crossfades (0.3s ease). No size jumps — the 40/60 split stays constant.
  • Progress indicator: Persistent at top of left panel — colored dots showing completed/current/upcoming phases
  • Touch targets: All buttons ≥60pt height. Answer options are full-width cards.
  • Pause button: Always visible at bottom of left panel across all phases
