Split-Screen Layout: How should the left panel change during the assessment?

The avatar stays on the right side throughout. The left panel changes as the patient progresses through each phase. Which layout approach feels right?

A

Minimal Left Panel — Avatar Leads

Left panel shows only: phase name, progress dots, and a small timer. The avatar does all the talking and guiding. Left panel is a thin sidebar (~30% width) so the avatar dominates. Clean, patient-focused.

Pros

  • Patient focuses on avatar, not screen
  • Feels like a real conversation
  • Less visual clutter for elderly patients

Cons

  • Clinician has less visibility into progress
  • No on-screen prompts if avatar audio is unclear
B

Rich Left Panel — Phase-Specific Content

Left panel (~45% width) shows different content per phase: orientation questions with checkmarks, word list for registration, clock drawing canvas, animal word chips for fluency, story text for logical memory. Each phase has a unique UI. Avatar still guides vocally.

Pros

  • Clinician can follow along and score
  • Visual reinforcement for hearing-impaired patients
  • Clock drawing canvas needs screen space anyway

Cons

  • More complex to build
  • Patient may look at screen instead of listening to avatar
C

Hybrid — Adaptive Panel

Left panel is minimal by default (phase name + progress), but expands with interactive content only when the phase requires it: clock drawing gets a canvas, verbal fluency gets a word counter, orientation gets answer buttons. Other phases stay minimal so the avatar leads.

Pros

  • Best of both — minimal when possible, rich when needed
  • Clock drawing and fluency NEED screen interaction
  • Keeps patient focused on avatar during listening phases

Cons

  • Inconsistent experience (panel changes size/content)
